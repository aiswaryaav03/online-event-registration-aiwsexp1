# Online Event Registration & Promotional Portal

## Aim
To design and develop an Online Event Registration & Promotional Portal using HTML5 form and media elements.

## Technologies Used
- HTML5
- CSS3

## Features
- Event banner image
- Promotional video
- Background audio
- Event registration form with validation

## Files Included
- index.html – Main webpage
- banner.jpg – Event banner image
- promo.mp4 – Promotional video
- intro.mp3 – Intro audio


## How to Run
1. Download or clone the repository
2. Open `index.html` in any web browser
